# Adjuvant

> An adjuvant is a substance added to a vaccine to boost the immune response to the antigen.

## Key facts

- Classical adjuvant: **aluminium hydroxide** (alum) — used since the 1920s.
- Modern adjuvants: **AS01 (shingles), MF59 (flu)**.
- Mechanism: mimic PAMPs → activate dendritic cells → bigger T-cell response.
- Trade-off: more side effects (sore arm, fever) for more durable immunity.
