# Dendritic Cell

> A dendritic cell is a sentinel that captures antigens in tissues and migrates to lymph nodes to teach T cells.

## Key facts

- The **most powerful APC** (antigen-presenting cell).
- Captures antigens in tissue, then travels via lymphatics to a lymph node — this journey takes ~1–2 days.
- Activates **naive T cells** — this is how adaptive immunity starts.
- Bridge between innate and adaptive systems.
