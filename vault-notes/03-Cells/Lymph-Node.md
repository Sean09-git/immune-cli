# Lymph Node

> A lymph node is a bean-shaped filter where dendritic cells meet T cells — where adaptive immunity is born.

## Key facts

- There are ~500–600 lymph nodes in the human body.
- Swollen lymph nodes = active immune meeting going on inside.
- Structure: cortex (B cell zone, germinal centres), paracortex (T cell zone), medulla.
- Filters lymph fluid — captures pathogens and tumour cells in transit.
