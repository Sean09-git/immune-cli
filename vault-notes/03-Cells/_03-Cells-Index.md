# Immune Cells

> The cast of cells. Each has a name, a job, and a place in the response timeline.

## Notes in this category

- [[Dendritic-Cell]] — *A dendritic cell is a sentinel that captures antigens in tissues and migrates to lymph nodes to teach T cells.*
- [[Mast-Cell]] — *A mast cell is a tissue-resident innate cell that releases histamine — the central player in allergy.*
- [[Plasma-Cell]] — *A plasma cell is a terminally differentiated B cell — an antibody factory.*
- [[Regulatory-T-Cell]] — *A regulatory T cell (Treg) is a subtype that brakes the immune response — preventing autoimmunity.*
- [[Lymph-Node]] — *A lymph node is a bean-shaped filter where dendritic cells meet T cells — where adaptive immunity is born.*

## How to read this folder

Start with the first note (the concept itself) and follow the [[links]] outward. Every note in this folder should connect to at least two others. If you find an isolated note, you've found something to fix.
