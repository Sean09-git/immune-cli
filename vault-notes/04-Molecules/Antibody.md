# Antibody

> An antibody (immunoglobulin) is a Y-shaped protein made by B cells that recognises one specific antigen.

## Key facts

- Five isotypes: **IgM** (first responder), **IgG** (most abundant in blood), **IgA** (mucosal), **IgE** (allergy), **IgD** (B-cell receptor).
- Functions: **neutralisation** (block pathogen), **opsonisation** (tag for phagocytes), **complement activation**.
- Each antibody has two arms (variable region = antigen binding) and a stem (constant region = function).
- Class switching changes the stem (IgM → IgG) without changing the binding site.

## App connection
