# Interferon

> An interferon is a cytokine that 'interferes' with viral replication and activates immune cells.

## Key facts

- Type I: **IFN-α, IFN-β** — secreted by virus-infected cells.
- Type II: **IFN-γ** — secreted by Th1, NK cells, CD8+ T cells. Activates macrophages.
- Type III: **IFN-λ** — recently discovered, mucosa-focused.
- Interferons make neighbouring cells temporarily virus-resistant — a real-time chemical alarm.

## App connection

Flu fatigue ≈ interferon overdose. Connect symptoms to the molecule causing them.
