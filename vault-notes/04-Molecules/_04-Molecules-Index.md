# Immune Molecules

> The chemistry of immunity. Cells talk and fight using these.

## Notes in this category

- [[Cytokine]] — *A cytokine is a small signalling protein that immune cells use to talk to each other.*
- [[Antibody]] — *An antibody (immunoglobulin) is a Y-shaped protein made by B cells that recognises one specific antigen.*
- [[Interferon]] — *An interferon is a cytokine that 'interferes' with viral replication and activates immune cells.*
- [[IgE]] — *IgE is the antibody class that drives allergic reactions by arming mast cells.*
- [[MHC]] — *MHC (Major Histocompatibility Complex) is a set of cell-surface molecules that display peptide fragments to T cells.*

## How to read this folder

Start with the first note (the concept itself) and follow the [[links]] outward. Every note in this folder should connect to at least two others. If you find an isolated note, you've found something to fix.
