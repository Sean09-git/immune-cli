# Immunodeficiency

> Immunodeficiency is a state of weakened or absent immune function — increasing infection susceptibility.

## Key facts

- **Primary** (genetic): SCID, Common Variable Immunodeficiency, IgA deficiency.
- **Secondary** (acquired): HIV/AIDS, chemotherapy, severe malnutrition, ageing.
- Pattern recognition: **'10 warning signs'** — frequent ear infections, sinusitis, unusual pathogens, etc.
- SCID famously fatal without bone marrow transplant — the 'bubble boy' condition.

## App connection

Frequent-infection pattern flagged with educational context — never a diagnosis.
