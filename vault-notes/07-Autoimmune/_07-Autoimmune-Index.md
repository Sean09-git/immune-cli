# Autoimmunity & Failure

> When the immune system attacks self, runs too weak, or runs too quiet.

## Notes in this category

- [[Autoimmunity]] — *Autoimmunity is a state where the immune system attacks the body's own tissues, mistaking 'self' for 'non-self'.*
- [[Type-1-Diabetes]] — *Type 1 diabetes is an autoimmune disease where T cells destroy the insulin-producing beta cells of the pancreas.*
- [[Multiple-Sclerosis]] — *Multiple sclerosis (MS) is an autoimmune disease where T cells attack the myelin sheath of nerve cells in the CNS.*
- [[Immunodeficiency]] — *Immunodeficiency is a state of weakened or absent immune function — increasing infection susceptibility.*
- [[Tolerance]] — *Immune tolerance is the failure-mode-avoiding property: the system learns NOT to attack self.*

## How to read this folder

Start with the first note (the concept itself) and follow the [[links]] outward. Every note in this folder should connect to at least two others. If you find an isolated note, you've found something to fix.
