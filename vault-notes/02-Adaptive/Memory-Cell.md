# Memory Cell

> A memory cell is a long-lived T or B cell that 'remembers' a past antigen and responds explosively on re-exposure.

## Key facts

- Two flavours: **memory T cells** (central, effector, tissue-resident) and **memory B cells**.
- Second-exposure response is **faster, stronger, and uses better antibodies** (IgG, not IgM).
- Lifespan: years to decades. Some smallpox memory cells persist 50+ years post-vaccination.
- Memory is the **single concept** that makes vaccination biologically possible.
