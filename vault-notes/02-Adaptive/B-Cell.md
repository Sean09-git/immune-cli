# B Cell

> A B cell is the antibody-producing arm of adaptive immunity — it becomes either a plasma cell or a memory cell.

## Key facts

- Matures in **bone marrow** (the 'B' is for bursa in birds, but bone marrow in humans).
- Activated B cells become **plasma cells** — antibody factories.
- Class switching: same B cell can change antibody type (IgM → IgG, IgA, IgE) under cytokine signals.
- Memory B cells live for decades — the basis of lifelong vaccine immunity.

## App connection

Antibody titer tests (e.g., post-vaccination check) directly measure your B cell output.
